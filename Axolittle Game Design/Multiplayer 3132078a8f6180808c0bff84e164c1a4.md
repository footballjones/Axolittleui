# Multiplayer

---

## Overview

All single-player systems are designed with multiplayer in mind — the lineage system and stat progression are built to extend naturally into social features without requiring a rebuild.

---

## Friend System

The foundation of all multiplayer features. Players add friends by username or invite link.

**Friend profiles show:**

- Current axolotl name, life stage, and color variant
- Character stats — Stamina, Strength, Speed, Intellect
- Current level and XP bar progress
- Generation number and lineage highlights
- Active aquarium preview

**Friends-only default:**
All social features default to friends-only. No interactions with strangers unless player explicitly opts in.

## Breeding with Friends

One of the most emotionally compelling and socially sticky features in the game. Both players’ axolotls contribute traits to produce two related-but-distinct eggs — one for each player.

**Requirements:**

- Both players must have an adult or elder axolotl
- Both players must agree to the breed — request and accept flow, never automatic
- Friends-only by default

**How it works:**

1. Player A sends a breeding request to Player B, selecting their axolotl
2. Player B receives notification and accepts, selecting their axolotl
3. Game combines traits from both parents — color tendencies, stat affinities, passive bonuses
4. Two eggs generated — one for each player. Related but not identical, like real siblings
5. Both eggs enter a 24-hour incubation period in each player’s nursery slot
6. Both players hatch and reveal results — shared anticipation drives conversation

**Trait combination:**
Weighted blend of parent traits with randomness. Prevents pure min-maxing while making high-lineage pairings genuinely more exciting. Occasionally a breeding produces a rare color variant or trait that neither parent had — these moments are shareable and memorable.

**Lineage display:**
The resulting axolotl’s profile shows both parent axolotls and their owners. Tapping a parent name shows that player’s profile. Creates natural social discovery.

**Cooldown:**
Once per 120 hour period. Feels special rather than spammable. Cooldown skippable with Opals.

**One-time XP boost:**
Breeding grants a one-time XP boost to th XP bar for that axolotl.

## Mini Game Competitions

Non Multiplayer mini games- Players never need to be online simultaneously. Same seed generates the same course for both players. Scores compared after both complete their run.

**How it works:**

1. Player challenges a friend to a specific mini game
2. Friend accepts — both receive the same seed / course configuration
3. Each player completes their run independently at any time within a 48-hour window
4. Scores compared and winner declared
5. Winner receives Bubble or Opal reward

**Available for:**

- Flappy Fish Hooks
- Keeper Upey
- Axolotl Stacker
- Math Rush
- Treasure Hunt Cave (see obstacle course below)

**Reward structure:**

- Winner — medium Coin reward
- Participant — small Coin reward for completing the challenge

## Treasure Hunt Cave

An extension of the Treasure Hunt Cave mini game built for competitive async play. The same seed-based cave generation used in single player is designed from day one to support this feature.

**How it works:**
Both players run the same procedurally generated cave course. Time and gem collection compared. Fastest time with most gems wins.

**Why it works:**

- No real-time synchronization required — async means no latency issues
- Reuses existing Treasure Hunt Cave code with a competitive layer on top
- Stat connection is clean — Speed and Strength both factor into course performance
- Results are shareable — players will post their run results

**Future extension:**
A weekly rotating course available to all players with a global leaderboard. Top performers earn exclusive cosmetic rewards.

## Leaderboards and Tiers

**Friend leaderboards:**
Ranked by overall growth stats, individual stat categories, and mini game high scores. Visible on the friend list screen.

**Global leaderboards:**
Optional opt-in. Top performers in weekly mini game competitions and obstacle course runs. Rewards exclusive cosmetic items for top tier finishers.

**Tiers:**
Seasonal competitive tiers based on mini game performance — Bronze, Silver, Gold, Diamond equivalent. Tier resets each season. Exclusive decoration reward for each tier achieved.

---

## Trading

**Decoration trading:**
Low risk and fun. Players can offer decorations to friends in exchange for other decorations or Coins. No Opal trading to prevent currency exploitation.

**Axolotl marketplace (future):**
Players list eggs for Coin or Opal bids. Rare color variants and high-lineage axolotls command premium prices. Creates a living community economy and organic discovery of exceptional animals.

## Age Rating Consideration

Breeding as a game mechanic has no age rating implications. Virtual creature breeding is a standard family-friendly mechanic used in Pokémon (rated E), My Singing Monsters (rated 4+), Dragon City (rated 4+), and Stardew Valley (rated E). Axolittle’s implementation — a request system producing an egg after a 24-hour period — is wholesome and clearly within 4+ territory.

## Build Considerations

Design these single-player systems now with multiplayer in mind:

| System | Multiplayer Extension |
| --- | --- |
| Lineage & trait inheritance | Breeding combines two players’ trait pools |
| Treasure Hunt seed generation | Same seed = same course for async competition |
| Character stats | Direct comparison surface for friend profiles |
|  |  |

Building these cleanly in single player means the multiplayer layer is an extension, not a rebuild.
